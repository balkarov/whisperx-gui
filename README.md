# WhisperX UI

Windows GUI-приложение для транскрипции видео и аудио на базе [WhisperX](https://github.com/m-bain/whisperX).

Просто запускаете `WhisperX-UI.exe`, указываете HuggingFace токен, выбираете файл, нажимаете "Старт" — получаете субтитры.
Установка Python **не требуется** — всё уже внутри.

## Возможности

- **17 форматов** аудио/видео: mp3, wav, flac, ogg, m4a, wma, aac, opus, mp4, mkv, avi, mov, wmv, webm, flv, ts, m4v
- **27 языков** (включая русский, английский, украинский) + авто-определение
- **6 моделей** Whisper: tiny, base, small, medium, large-v2, large-v3
- **6 форматов вывода**: SRT, VTT, TXT, TSV, JSON, ASS
- **Диаризация** — определение спикеров (кто говорит)
- **Пакетная обработка** — выберите папку и обработайте все файлы разом
- **Перевод на английский** — встроенный режим translate
- **GPU-ускорение** — автоматическое определение NVIDIA GPU
- **Сохранение настроек** между сессиями

## Быстрый старт (для пользователей)

1. Скачайте и распакуйте архив `WhisperX-UI.zip`
2. Запустите `WhisperX-UI.exe`
3. Получите HuggingFace токен (см. инструкцию ниже) и вставьте в поле "HuggingFace токен"
4. Выберите видео/аудио файл
5. Нажмите "Старт"
6. При первом запуске модели скачаются автоматически (нужен интернет)
7. Субтитры будут сохранены рядом с исходным файлом

## Модели

| Модель | Размер | Качество | Скорость |
|--------|--------|----------|----------|
| tiny | ~75 МБ | Низкое | Очень быстрая |
| base | ~145 МБ | Ниже среднего | Быстрая |
| small | ~480 МБ | Среднее | Средняя |
| **medium** | **~1.5 ГБ** | **Хорошее** | **Средняя** |
| large-v2 | ~3 ГБ | Отличное | Медленная |

Все модели скачиваются автоматически при первом использовании. После скачивания хранятся в кеше и работают без интернета.

## Как получить HuggingFace токен

Токен нужен для скачивания всех моделей (транскрипции и диаризации). Указывается один раз — дальше сохраняется в настройках.

### Пошаговая инструкция

#### Шаг 1: Регистрация на HuggingFace

1. Откройте сайт [huggingface.co](https://huggingface.co)
2. Нажмите **"Sign Up"** в правом верхнем углу
3. Введите email, придумайте пароль, выберите имя пользователя
4. Подтвердите email (придёт письмо на почту)

#### Шаг 2: Создание токена

1. Войдите в свой аккаунт на [huggingface.co](https://huggingface.co)
2. Нажмите на свой **аватар** (иконку) в правом верхнем углу
3. Выберите **"Settings"** (Настройки)
4. В левом меню нажмите **"Access Tokens"**
   - Или перейдите напрямую: [huggingface.co/settings/tokens](https://huggingface.co/settings/tokens)
5. Нажмите кнопку **"Create new token"**
6. Заполните:
   - **Token name** — любое имя, например `whisperx`
   - **Token type** — выберите **"Read"** (чтение) — этого достаточно
7. Нажмите **"Create token"**
8. **Скопируйте токен** — он начинается с `hf_...`

> **Важно!** Токен показывается только один раз. Если потеряете — придётся создать новый.

#### Шаг 3: Принять лицензионные соглашения моделей

Модели диаризации (определение спикеров) требуют принятия лицензии на сайте HuggingFace. **Без этого шага диаризация не будет работать.**

1. Откройте страницу модели [pyannote/segmentation-3.0](https://huggingface.co/pyannote/segmentation-3.0)
   - Вы увидите раздел **"Gated model"** с текстом лицензии
   - Заполните имя и нажмите **"Agree and access repository"**
   - Дождитесь подтверждения (обычно мгновенно)

2. Откройте страницу модели [pyannote/speaker-diarization-3.1](https://huggingface.co/pyannote/speaker-diarization-3.1)
   - Аналогично: заполните и нажмите **"Agree and access repository"**

> **Если не принять соглашения**, приложение выдаст ошибку при попытке диаризации. Транскрипция (без спикеров) будет работать и без этого шага.

#### Шаг 4: Вставить токен в приложение

1. Запустите **WhisperX UI**
2. Включите галочку **"Включить диаризацию"**
3. В поле **"HuggingFace токен"** вставьте скопированный токен (Ctrl+V)
4. Нажмите **"Старт"** — модели скачаются автоматически при первом запуске
5. Токен сохраняется — повторно вводить не нужно

#### Что если я не хочу регистрироваться?

Без токена приложение **не сможет скачать модели**. Это ограничение авторов моделей, а не приложения. Регистрация бесплатна и занимает 2 минуты.

## Где хранятся скачанные модели

Модели скачиваются в кеш HuggingFace:

```
%USERPROFILE%\.cache\huggingface\hub\
```

Например: `C:\Users\Иван\.cache\huggingface\hub\`

Если нужно изменить расположение кеша, задайте переменную окружения перед запуском:

```
set HF_HOME=D:\models\huggingface
```

## Настройки приложения

Настройки (модель, язык, токен и пр.) сохраняются автоматически:

```
%APPDATA%\WhisperX-UI\config.json
```

## Для разработчиков

### Установка из исходников

```bash
# Python 3.10+
pip install -r requirements.txt

# FFmpeg
winget install Gyan.FFmpeg

# Опционально: PyTorch с CUDA для GPU
pip install torch torchaudio --index-url https://download.pytorch.org/whl/cu118
```

### Запуск

```bash
python app.py
```

### Сборка exe

```bash
pip install pyinstaller
pyinstaller build.spec --noconfirm
```

Результат: папка `dist/WhisperX-UI/` с `WhisperX-UI.exe`.

### Структура проекта

```
whisperx-ui/
├── app.py                 # Точка входа
├── build.spec             # Конфигурация PyInstaller
├── requirements.txt       # Зависимости
├── README.md
├── core/
│   ├── __init__.py
│   ├── transcriber.py     # Логика транскрипции (WhisperX pipeline)
│   └── utils.py           # Утилиты, константы, сохранение результатов
└── ui/
    ├── __init__.py
    └── main_window.py     # GUI (CustomTkinter)
```
