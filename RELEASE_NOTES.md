## WhisperX UI v1.0.0

**Windows GUI-приложение для транскрипции видео и аудио с определением спикеров.**

Просто запустите `WhisperX-UI.exe` — установка Python не требуется.

### Возможности

- 🎙 **Транскрипция** видео и аудио на базе WhisperX
- 👥 **Диаризация** — определение кто говорит (спикеры)
- 🌍 **27 языков** включая русский, английский, украинский + авто-определение
- 📁 **17 форматов** входных файлов (mp4, mkv, mp3, wav, flac и др.)
- 📝 **6 форматов** субтитров: SRT, VTT, TXT, TSV, JSON, ASS
- 🚀 **6 моделей** Whisper: tiny, base, small, medium, large-v2, large-v3
- 📂 **Пакетная обработка** — выберите папку и обработайте все файлы
- 🔄 **Перевод на английский** — встроенный режим translate
- ⚡ **GPU-ускорение** — автоматическое определение NVIDIA GPU
- 💾 **Сохранение настроек** между сессиями

### Как начать

1. Скачайте `WhisperX-UI.zip` из раздела Assets ниже
2. Распакуйте в любую папку
3. Запустите `WhisperX-UI.exe`
4. Укажите HuggingFace токен ([как получить](https://github.com/balkarov/whisperx-gui#как-получить-huggingface-токен))
5. Выберите файл и нажмите «Старт»

> ⚠ **При первом запуске** модели скачиваются автоматически (нужен интернет). После этого работает оффлайн.

### Системные требования

- Windows 10/11 (64-bit)
- 4 ГБ ОЗУ (рекомендуется 8+ ГБ)
- ~2 ГБ свободного места (приложение + модель medium)
- Интернет для первого скачивания моделей
- NVIDIA GPU (опционально, для ускорения)
